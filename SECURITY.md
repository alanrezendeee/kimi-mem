# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please email us at **security@theretech.dev** (or open a private issue if enabled).

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work on a fix promptly.

## Privacy Notes

- kimi-mem stores data locally in SQLite
- Memories tagged with `<private>` are excluded from search
- No telemetry or external data collection
